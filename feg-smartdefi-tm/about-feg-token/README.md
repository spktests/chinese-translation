### FEG：DeFi 的未来

FEG 代币是 SmartDeFi 生态系统的治理代币，旨在通过其创新的代币经济学和强大的生态系统彻底改变去中心化金融 (DeFi) 格局。

FEG 代币在 BNB 链上运行，总供应量为 1000 亿。
